1. Download: **[Latest Version](https://github.com/whtmst/t-oom/archive/master.zip)**
2. Unpack the Zip file `t-oom-main.zip`
3. Find `T-OoM` folder inside `t-oom-main` folder
4. Copy `T-OoM` folder into \Interface\AddOns
5. Restart game
